# TeleHealth AI

AI-powered virtual doctor app with symptom checker, appointment booking, and telehealth consultations.

## Features

- **AI Symptom Checker**: Describe your symptoms for AI-powered insights
- **Appointment Booking**: Schedule video or in-person consultations
- **Health Records**: Store and manage medical records
- **Telehealth Consultations**: Video calls with healthcare providers
- **Medication Tracking**: Manage prescriptions

## Tech Stack

- Expo SDK 54
- React Native 0.79
- expo-router (file-based navigation)
- RevenueCat (subscription monetization)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npx expo start
```
